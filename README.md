# Camp portal

## О проекте

**Camp portal** - учебный веб-проект информационной системы управления детским оздоровительным лагерем.

---

## Краткое описание

Система предназначена для автоматизации базовых процессов детского лагеря:

- учет детей;
- учет сотрудников;
- формирование отрядов;
- создание мероприятий;
- отметка участия детей;
- формирование простых отчетов;
- отображение ближайших событий.

---

## Стек технологий

- **Python 3.12** — основной язык серверной логики.
- **FastAPI** — высокопроизводительный асинхронный веб-фреймворк для REST API.
- **PostgreSQL** — надежная реляционная СУБД для хранения данных.
- **SQLAlchemy (ORM)** — связующее звено между базой данных и объектами Python.
- **Vanilla JS** — клиентская часть (SPA), реализованная на нативном JavaScript.
- **JWT (JSON Web Token)** — безопасная аутентификация и хранение сессий в LocalStorage

---

## Минимальный функционал

- Авторизация пользователей
- Главная панель
- Список детей
- Добавление и редактирование детей
- Список сотрудников
- Список отрядов
- Список мероприятий
- Отметка участия детей
- Отчет по активности детей
