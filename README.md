# Camp Management System

Учебный веб-проект информационной системы управления детским оздоровительным лагерем.

## Краткое описание

Система предназначена для автоматизации базовых процессов детского лагеря:

- учет детей;
- учет сотрудников;
- формирование отрядов;
- создание мероприятий;
- отметка участия детей;
- формирование простых отчетов;
- отображение ближайших событий.

Проект реализуется как учебный MVP, то есть минимальная рабочая версия системы.

## Стек технологий

- Python
- Flask
- Jinja2
- PostgreSQL
- psycopg2
- HTML/CSS
- Git/GitHub
- Render

## Минимальный функционал

- Авторизация пользователей
- Главная панель
- Список детей
- Добавление и редактирование детей
- Список сотрудников
- Список отрядов
- Список мероприятий
- Отметка участия детей
- Отчет по активности детей
