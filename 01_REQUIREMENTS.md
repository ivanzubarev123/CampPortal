# План разработки проекта

## 1. Цель проекта

Разработать учебный веб-проект информационной системы управления детским оздоровительным лагерем.

Система должна демонстрировать:

- работу веб-приложения;
- подключение к базе данных;
- авторизацию пользователей;
- хранение и отображение данных;
- базовую отчетность.

## 2. Формат проекта

Проект выполняется как MVP.

1. Пользователь входит в систему.
2. Добавляет или просматривает детей.
3. Создает мероприятие.
4. Отмечает участие детей.
5. Получает отчет.

## 4. Минимальный критерий готовности

Проект считается готовым, если:

- сайт открывается по ссылке;
- работает вход;
- данные берутся из бд;
- можно добавить ребенка;
- можно создать мероприятие;
- можно отметить участие;
- отчет показывает данные;
- есть документация;
- есть отчет.
